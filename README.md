# walkcyclejs
A web-based, customizable walkcycle animation. <br />
Taking the [css3walkcycle](https://github.com/KarstenBuckstegge/css3walkcycle) to new heights.

## Tools used
- npm only build process
- babel for ES6
- SCSS with libsass

## Getting started
```  
nvm use v0.12.6
npm install
npm start
```
